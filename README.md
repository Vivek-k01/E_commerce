# 🛒 Apni Shop - Full Stack E-Commerce App

A professional web application built using **Spring Boot**, **MySQL**, and **Thymeleaf**. This project follows a clean **Service Layer** architecture.

## ✨ Key Features
* **Admin Dashboard:** Manage products (Add, Edit, Delete) with image uploads.
* **Shopping Cart:** Fully functional cart with 'Buy Now' and item removal.
* **Professional Architecture:** Separate Service layer for business logic.
* **Order Management:** Track customer orders and details.
* **PDF Invoices:** Automated receipt generation using **iText**.
* **WhatsApp Integration:** Direct admin reply and order confirmation via WhatsApp.

## 🛠️ Technology Stack
* **Backend:** Java Spring Boot 3, Maven
* **Frontend:** Thymeleaf, HTML5, CSS3, Bootstrap 5
* **Database:** MySQL
* **Tools:** VS Code, Git

## 🚀 How to Run locally
1. Clone this repository.
2. Setup a MySQL database named `ecommerce_db`.
3. Update `src/main/resources/application.properties` with your MySQL username and password.
4. Run the command: `./mvnw spring-boot:run`
